# OOH Sports Advertising Analysis

##  Overview
This project analyzes out-of-home (OOH) advertising during live sports events, focusing on brand exposure, placement strategies, and industry trends.

The dataset captures real-world observations of advertisements during live sporting events, including brands, industries, placement types, and frequency of exposure.

---

##  Problem Statement
Brands invest heavily in sports advertising, but understanding which placements and industries dominate visibility is critical for optimizing marketing strategies. This project explores how advertising exposure varies across sports, locations, and placement types.

---

##  Objectives
- Identify top brands with highest visibility
- Analyze industry-level advertising dominance
- Evaluate effectiveness of placement types
- Compare in-venue vs broadcast advertising exposure
- Understand brand distribution across sports

---

##  Dataset Description
The dataset includes observed advertising data from live sports events.

Key features:
- Event / Game (e.g., NFL, NBA games)
- Sport type
- Venue and location
- Brand name
- Industry (Apparel, Automotive, Beverages, etc.)
- Ad placement type (scoreboard, signage, commercials)
- Placement location (in-venue, broadcast, digital)
- Frequency observed

---

##  Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

##  Key Analysis Performed
- Top brands by advertising frequency
- Industry-level advertising comparison
- Ad placement type analysis
- Placement location distribution
- Cross-analysis (brand by sport, industry by sport)
- Data cleaning and validation

---

##  Key Insights
- Major brands such as Nike, Adidas, Pepsi, and Toyota consistently appear across events
- Industries like apparel, automotive, and beverages dominate sports advertising
- In-venue advertising has the highest exposure frequency compared to broadcast
- Placement types such as commercials and signage drive visibility
- Brand exposure is evenly distributed across major sports leagues
